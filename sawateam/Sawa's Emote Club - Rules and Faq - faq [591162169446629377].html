How do I read manhwa/manga/manhua in this server?
Chapters are posted on our site https://sawateam.info/ which are linked in our published updates channel.
I can't access the site! It keeps saying you need to verify your discord account.
As long as you're in this server, you should be able to read any of the chapters. There is a button to verify. Click on that button, and it should lead you to a screen asking to see what servers you're in. This is so the website can check that you're in this server and grant you access. If it's not working, please ask in #help-with-chapters or try again on a different browser. Please wait 1 minute if it keeps redirecting you to the verification page, as there are known issues with caching.
I can't see #deleted-channel!
Read to the end of this channel to see how to access the channel.
I can't see images on your website at all!
We host some of our chapters on Imgur. Please check to see if your country has blocked Imgur. If so, then you'll have to use a VPN to view our chapters.
I want to read your chapters offline on apps like Tachiyomi. How do I do that?
This is not possible because of the way the website is built. If it's website exclusive, please consider buying the official on Tapas, TappyToon, Pocket Comics, Inkr, etc. so you can read offline on your phone.
Why are you asking to access my Discord account to read your chapters? This is suspicious. Are you selling my information?
We used to use passwords, which was a huge headache. Discord verification is a shortcut for that since people had to be in our server anyway to get the passwords. It protects ourselves if anything goes wrong with the authors/publishers. We only ask for a list of servers, and there’s no way to connect the user to the list of servers since we don’t ask for email or username or name, so no user data/information is ever stored. If you'd like more clarification on how this works, please DM me.

Are images not loading on the website? Are all images appearing as white boxes or something similar? Read through these instructions for fixes. Most of the time, these issues are on the client side, meaning that you can usually fix these issues by changing settings etc. ️ iPhone/iPad specific fixes: 1) Do not read from the pop-up safari browser when you click on it from the discord app. Go to the actual browser. Safari dislikes loading multiple images and will purposefully redirect or load the white squares as you see in some of your screens. Use the Chrome app. 2) Go to settings > scroll down to either SAFARI or CHROME (or both) > turn OFF "prevent cross-site tracking". If none of these work, please access chapters on your laptop/computer instead. ️ Laptop/Computer Fixes: 1) Use incognito mode or a VPN - Windscribe is recommended. 2) Clear your cache. 3) If you're using the Safari browser, change to either Chrome or Firefox. 4) Check your wifi connection and make sure it's not slow. 5) Turn off toolbars and extensions. 6) Turn on Javascript. 7) Turn off adblock. (Don't worry, we don't have ads on the website) ️ Other: Try the steps on this website if you're still having issues https://www.partitionwizard.com/partitionmagic/images-not-showing-in-google-chrome-024.html If all else fails, try uninstalling and reinstalling your browser or reset to default settings.
